## Overview
A terminal-based snake game built with C, featuring smooth gameplay, real-time controls, and a retro text interface.

## How to play
* **Objective**: Eat food to grow the snake.
* **Avoid collisions** with your own tail to keep playing.


## Run the Game
```bash
gcc -o snake main.c
./snake
```

## Controls
* `Esc`: Quit
* `W`: Move up
* `S`: Move down
* `A`: Move left
* `D`: Move right
